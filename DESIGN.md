# BIT://GENESIS — 创世 MVP 设计便签

> 与旧版 (cs_open_world/index.html 引擎v3) 并列的全新极简原型，旧版不动。
> 世界观：修复重启一台濒死的旧电脑。每学一个知识点，世界多一种能力，UI 系统跟着诞生。
> 边玩边拓展；本文件是拓展时的地图。

## 语言法则（用户 2026-07-04 立法，覆盖一切文案）
**世界里能显示什么文字，取决于玩家解锁了什么编码。**
1. ASCII 之前：**无字母无句子**——独白也不许有。教学全靠演示、图形、闪烁；数字/符号(0 1 ✕ : ▸)随"数"诞生，不受限。
2. ASCII 之后：文字合法，但**只有英文**（独白/回声/路面全英文）。
3. Unicode 之后：**中文诞生**（豆腐块▯▯解析成"你好"的瞬间）——之后独白切回母语，章末卡才合法。
4. 键帽(E/方向键)是 meta-UI：键盘属于现实世界，不受法则限制，但必须画成键帽形。

## 核心谜题哲学：无言归纳（用户 2026-07-04 定调）
理想形态是 The Witness 式"找规律"：**不给面板、不给说明、不给目标数字**——
世界自己把规律演给你看（灯序列按二进制数数），停住，等你摆出下一个。
悟出规律的瞬间 = 学会知识点的瞬间，两者是同一件事。
- 已实现：二进制三轮序列（数数进位 → 大进位 → 等差×3）。
- 无言提示阶梯（替代文字提示）：摆错→轻红+序列重播；错2次→重播时琥珀高亮"翻动的位"；错4次→才出一行独白。
- 待改造：BCD（观察数字组"数到9就归零"的异常规律）、ASCII（罗塞塔石碑式对照残页，删说明文字）。
- 文字只在两处合法出现：成功后的独白（奖励叙事）、多次失败后的兜底提示。

## 铁律（来自 CS游戏调研 + 教育游戏循证）
1. **开局屏幕上什么都没有** —— 信息过载在结构上不可能：一切 UI/系统都是玩家造出来的。
2. **一次只教一个机制**（Nintendo 三段式）；60 秒内第一次成功。
3. **一屏可交互物 ≤3**（BotW 三角形法则）；同一时刻只有一个发光地标。
4. **诱人细节测试**：任何元素若不服务知识点，删。叙事只用内心独白+回声，不放石碑 lore。
5. **提示阶梯不给答案**：概念 → 用法 → 同构例题（换数字）。
6. **世界还没有声音**——这是设定（SAMPLING 章之前保持静音），不是偷懒。

## 后续拓展路线（按创世纪主线，每拍一个 commit 的量）
1. **PIXEL/分辨率**：01 雨聚合成粗像素格 → 分辨率逐级提升演出；小地图系统诞生。
2. **COLOUR DEPTH**：黑白→彩色；随即触发**存储警告**（文件大小=分辨率×位深，算式亮出来）。
3. **RLE**：压缩天空等大片纯色，警告解除；★挑战：压花丛发现压不动（RLE 失效面）。
4. **VECTOR**：换一种根本不占空间的表示；画图小玩法。
5. **SOUND/采样率**：世界第一次有声音+BGM；第二次存储警告 → lossy 钩子。
6. 网络 → 逻辑门(nandgame 节奏) → CPU → 汇编 → OS(背包/存档UI诞生) → 安全 → SQL(排行榜诞生+citizens 表通真结局)。

## 代码拓展点 (index.html)
- `BEATS`：章节状态机（存档恢复用检查点）。
- `LANDMARKS`：世界地标（可见性由 stage 决定，保持"同屏一个发光地标"）。
- 谜题 = `openPanel(html)` + 自动判定（达成即过，无提交按钮）+ `hintLadder(key, [三级提示])`。
- 世界渲染在 `draw()` 里按 `G.unlocked.*` 分层叠加（学了什么，世界才画什么）。
